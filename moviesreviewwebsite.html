Category: Web Development

Language: PHP, MySQl, HTML, CSS, Javascript

Year: 2023

Description: Introduction:
This website allows users to view and review movies, as well as view the trailer of the movie. It is also able to check who is logged in using cookies. Login details can also be saved for easier login when a user logs out. Cybersecurity features such as hashing and prevention of SQL injections are also being implemented in the website.

Key Features:
- User registration & login
- User who are registered can login to their account to update their information
- Display list of movies & details
- Users are able to write/edit/delete their own reviews [only for registered users]
- Light/Dark mode, allows users to choose their preferred mode for viewing

Development:
This website was built using PHP, MySQL, HTML, CSS, and JavaScript. It includes features such as user registration and login functionalities, allowing users to securely create accounts, log in, and manage their personal information. Registered users can update their details and choose to edit or delete their accounts. Additionally, the site displays a list of movies from a database, presenting key information like movie title, poster image, genre,, and trailer. Furthermore, users can access movie reviews, including details like the username of the reviewer, review content, posting date, and movie rating.
