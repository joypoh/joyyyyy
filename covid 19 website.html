Category: Web Development

Language: HTML, CSS, Javascript

Year: 2023

Description: Introduction:
This website allows users to view the current status of Covid-19 in Singapore, such as total number of cases, number of people currently in hospital and the total number of deaths. It also provides insights to the breakdown of the cases. Users are also able to view the progression of the vaccination status in Singapore, as well as the various locations of the vaccination centres in Singapore and the type of vaccine administrated as eacch locations.

Key Features:
- Overview on Covid-19 Cases
- Vaccination Progression status
- List of Vaccination Centres with the respective location address, vaccination type & phone number
- Includes search function to search for vaccination locations available

Development:
This website was built using HTML, CSS and Javascript. The libraries used in this projects were anime.js for the animations, fullPage.js and Bootstrap for the layout.
